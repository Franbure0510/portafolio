# 🚀 Portfolio - François Larrabure

Portfolio personal desarrollado con **Flask** (Python) + **HTML, CSS, JavaScript**.

## ✨ Características

- Diseño moderno y responsive
- Animaciones suaves al hacer scroll
- Formulario de contacto funcional
- Certificaciones con enlaces verificables
- Optimizado para PDF (CV descargable)

## 🛠️ Tech Stack

| Frontend | Backend | Deployment |
|----------|---------|------------|
| HTML5 | Python 3 | Render |
| CSS3 | Flask | GitHub |
| JavaScript | Gunicorn |  |

## 🚀 Live Demo

[https://portafolio.onrender.com](https://portafolio.onrender.com)

## 📦 Instalación local

```bash
git clone https://github.com/Franbure0510/portafolio.git
cd portafolio
pip install -r requirements.txt
python app.py
```

Abrir [http://localhost:5000](http://localhost:5000)
